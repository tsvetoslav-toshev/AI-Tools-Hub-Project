# Design System - Modern Minimalist

**Project Style Guide for vibecode-full-stack-starter-kit**

## Design Philosophy

**Vibe:** Luxury simplicity — typography is the hero  
**Approach:** Lots of white space, thin lines, muted palette, refined typography, calm simplicity, soft shadows

## Color Palette

### Primary Colors (Monochrome + Soft Neutrals)
- **Pure White:** `#FFFFFF` - Main background
- **Soft Gray:** `#F4F4F4` - Secondary background, subtle sections
- **Medium Gray:** `#A3A3A3` - Secondary text, borders
- **Deep Black:** `#1A1A1A` - Primary text, headlines

### Usage Guidelines
- Use `#FFFFFF` for main backgrounds
- Use `#F4F4F4` for cards, sections, subtle separations
- Use `#A3A3A3` for secondary information, hints, borders
- Use `#1A1A1A` for primary content, headlines, important text

## Typography

### Font Hierarchy
- **Headlines:** Large, bold, generous spacing
- **Body Text:** Clean, readable, comfortable line height
- **Secondary Text:** Lighter weight, smaller size

### Best Practices
- Typography is the hero - let it breathe
- Generous letter-spacing for headlines
- Comfortable line-height (1.6-1.8) for body text
- Minimal font weights (light, regular, medium)

## Spacing

### White Space Philosophy
- **Embrace emptiness** - white space is a design element
- Large padding/margins between sections
- Generous spacing around text elements
- Don't fill every pixel

### Scale
- Base unit: 8px
- Small: 16px (2 units)
- Medium: 32px (4 units)
- Large: 64px (8 units)
- Extra Large: 96px+ (12+ units)

## Components

### Borders & Lines
- **Thin lines only** - 1px maximum
- Subtle colors: `#F4F4F4` or `#A3A3A3`
- Use sparingly for gentle separation

### Shadows
- **Soft and subtle only**
- Example: `0 2px 8px rgba(0, 0, 0, 0.04)`
- Avoid harsh, dark shadows

### Buttons
- Minimal style: thin border or simple background
- Generous padding
- Clear hover states (subtle)
- No heavy effects or gradients

### Cards
- Clean, simple rectangles
- Soft background (`#F4F4F4`)
- Minimal or no border
- Soft shadow if needed

## Dark Mode

### Inverted Palette
- **Deep Black:** `#1A1A1A` - Main background
- **Dark Gray:** `#2A2A2A` - Secondary background
- **Medium Gray:** `#A3A3A3` - Secondary text
- **Pure White:** `#FFFFFF` - Primary text

### Principles
- Maintain the same minimalist approach
- Ensure sufficient contrast
- Keep soft shadows (lighter, not darker)

## What to Avoid

❌ Bright, saturated colors  
❌ Heavy shadows or effects  
❌ Cluttered layouts  
❌ Multiple competing elements  
❌ Unnecessary decorations  
❌ Thick borders (>1px)  
❌ Busy backgrounds or patterns

## What to Embrace

✅ White space as a design element  
✅ Clean, refined typography  
✅ Subtle, intentional accents  
✅ Calm, focused layouts  
✅ Gentle transitions  
✅ Breathing room between elements

---

**Remember:** Less is more. Every element should have a purpose. Let simplicity and typography lead the design.
